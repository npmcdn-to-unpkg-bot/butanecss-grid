# ButaneCSS Grid

## Getting Started
```shell
git clone https://github.com/ButaneCSS/butane-grid.git
cd butane-grid
npm i
gulp
```

## Usage
```html
<div class="o-grid o-grid--gutters">
	<div class="o-grid__item u-w6@sm">
		<p>Example grid item text.</p>
	</div>
	<div class="o-grid__item u-w6@sm">
		<p>Example grid item text.</p>
	</div>
	<div class="o-grid__item u-w6 u-w3@sm">
		<p>Example grid item text.</p>
	</div>
	<div class="o-grid__item u-w6 u-w3@sm">
		<p>Example grid item text.</p>
	</div>
	<div class="o-grid__item u-w6 u-w3@sm">
		<p>Example grid item text.</p>
	</div>
	<div class="o-grid__item u-w6 u-w3@sm">
		<p>Example grid item text.</p>
	</div>
</div>
```
